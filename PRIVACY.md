# Privacy Policy

Effective date: May 28, 2026

Excalicon does not collect, store, transmit, sell, or share personal data.

## Data Collection

Excalicon does not collect any personal information, browsing history, Excalidraw document contents, search queries, drawings, clipboard contents, or usage analytics.

## Data Storage

Excalicon does not store user data. Icon search text, fill selection, and color selection are used only in the local side panel while the extension is open.

## Network Requests

Excalicon does not call external APIs or analytics services. The Material Symbols SVG files are packaged with the extension.

## Permissions

Excalicon uses Chrome’s `sidePanel` permission to display the searchable icon panel.

Excalicon requests access to `https://excalidraw.com/*` and `https://*.excalidraw.com/*` only to enable drag and drop into Excalidraw. The injected script converts icons dragged from the extension into SVG data that Excalidraw can import at the drop location.

## Third-Party Assets

Excalicon includes Google Material Symbols SVG assets from the `@material-symbols/svg-400` package. Material Symbols are distributed under the Apache License 2.0.

## Contact

For privacy questions, open an issue in the public Excalicon GitHub repository.
